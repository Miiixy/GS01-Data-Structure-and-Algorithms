# Sistema de Monitoramento Espacial

## Descrição

Projeto desenvolvido em linguagem C com o objetivo de simular o monitoramento de uma missão espacial.

O sistema permite cadastrar informações da nave, visualizar o status dos sensores, realizar análises automáticas, simular leituras e consultar um histórico de dados registrados.

## Como Executar

1. Abra o projeto em uma IDE compatível com C.
2. Compile o arquivo `main.c`.
3. Execute o programa.
4. Utilize o menu para acessar as funcionalidades.

---

## Estrutura do Sistema

### 1 - Inserir Dados

Permite cadastrar temperatura, energia e status da comunicação.

### 2 - Visualizar Status

Exibe os dados atualmente registrados.

### 3 - Executar Análise

Analisa os parâmetros da missão e identifica possíveis alertas.

### 4 - Simular Sensores

Gera leituras aleatórias dos sensores e as registra no histórico.

### 5 - Histórico de Leituras

Exibe todas as leituras armazenadas e calcula médias dos dados coletados.

### 6 - Encerrar

Finaliza a execução do sistema.
